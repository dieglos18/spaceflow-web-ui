# Use of AI in this project

This document describes how AI tools were used during the development of SpaceFlow Web UI.


## Role of the developer

All product and technical decisions were made by me. I defined the stack, chose the features to build, and then planned each one so that Cursor could implement them. The AI tools acted as an implementation assistant; direction and priorities were my responsability.


## Cursor

**Cursor** was used for day-to-day development:

- Implementing features (responsive tables, Spaces carousel padding, Docker setup, etc.)
- Refactoring (e.g. Place detail into folder with modals and sections)
- Fixes and small changes (removing unused pages, README updates, admin login layout)
- Guidance and planning when requested


## ChatGPT

**ChatGPT** was used to:

- **Define the color palette** — Choice of brand and UI colors (primary, surfaces, semantic colors, neutrals) and the light/dark theme variables used in `src/index.css`.


## Summary

| Tool     | Use |
|----------|-----|
| Cursor   | Implementation, refactors, fixes, planning |
| ChatGPT  | Color palette and design tokens |
